# eslint-config-doctolib

ESLint configuration used in Doctolib projects.

## Install

```
npm install eslint-config-doctolib
```

## Usage

### Basic

```js
// .eslintrc
{
  "extends": "doctolib"
}
```

### Test

```js
// .eslintrc
{
  "extends": "doctolib/test"
}
```

### React

```js
// .eslintrc
{
  "extends": "doctolib/react"
}
```
